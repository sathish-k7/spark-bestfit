spark-bestfit
==============

Modern Spark distribution fitting library with efficient parallel processing.

Automatically fit ~100 scipy.stats distributions to your data using Apache Spark's
distributed computing power, with optimized Pandas UDFs.

**Supported Versions:**

- Python 3.11 - 3.13
- Apache Spark 3.5.x and 4.0.x
- See :doc:`quickstart` for the full compatibility matrix
